# Atera RMM Introduction

Atera RMM is a cloud-based Remote Monitoring and Management (RMM) platform used by IT professionals and Managed Service Providers (MSPs) to monitor, manage, and support computers and networks remotely from a single dashboard.

## Key Features

Atera combines multiple tools into one platform, including:

* **Remote Monitoring**
  - Track device health  
  - Detect issues early  
* **Remote Access**
  - Integrates with tools like Splashtop and AnyDesk, TeamViewer. 
* **Patch Management**
  - Windows updates  
  - Software patching  
* **Scripting & Automation**
  - Run scripts  
  - Automate repetitive tasks  
* **Helpdesk / Ticketing (PSA Features)**
  - Receive and manage support requests  

This allows IT teams to efficiently support multiple users and devices without needing to be physically present.
